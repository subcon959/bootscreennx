About
What is this?

This is a tool to generate Old School BIOS images that emulate classic PCs; it is intended to be used on the Nintendo Switch. Currently the only method to display a splash screen is to use Hekate and enable autoboot.

How do I use this?
  1. Tweak the above settings to your liking to generate your custom splash
  2. Download the PNG file
    3. To convert the file for use on hekate:
  3. Convert the image to a 32-bit (ARG8) BMP
  4. Rotate the image 90 degrees counter-clockwise
  5. Save the file as "bootlogo.bmp" and save it in the booloader/ folder on your SD root
    6. More information about Hekate's policy on loading a bootlogo can be found here
